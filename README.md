# Rocnikovka
Moje ročníkovka - E-Shop

Co funguje:
  načítání produktů
  vybírání mezi produkty
  menu (výběr mezi nákupem, košíkem, účtem či vypnutím)
  obrázky produktů
  
Na čem se pracuje:
  Single Item Display
  revamp produktů aby byli na special classe
  
Co není:
  košík
  objednávání
  účty
